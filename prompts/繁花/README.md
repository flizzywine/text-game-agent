# 繁花 Prompt Profile

基于 `prompts/基准/` 的副本改造，目标是测试繁花预设里较有效的花语控制信号。

保留：
- 花店 / Hana / 花语 / 顾客的叙事隐喻
- Hana 只翻译花语，不进入预言故事
- 花语只占卜近期可能走向
- 顾客无叙事特权，需求仍需符合预言
- 防抢话
- 防转述
- 防重复
- 防八股
- 防机械
- 角色状态自查
- 段落和标点纪律

舍弃：
- 大量最高优先级宣称
- COT 标签
- 酒馆变量和正则脚本
- UI 美化 HTML
- 固定 800-1000 字要求

启用方式：

```text
把 prompts/active-profile.txt 改成：繁花
```

回退方式：

```text
把 prompts/active-profile.txt 改成：基准
```
